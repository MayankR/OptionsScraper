Use init_db.ipynb to initial a local MySQL DB.
Use ScrapeYahoo.ipynb to scrape SPY options data from Yahoo Finance and load it into a table.

Information loaded in DB:
- contract name
- ticker symbol
- type (call or put)
- strike price
- last price
- bid price
- ask price
- vol
- iv
- expiry date
